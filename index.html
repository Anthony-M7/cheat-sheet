Guía Rápida de Bit & Vatio: Flexbox vs Grid

🧩 FLEXBOX

El Contorsionista: Ideal para alinear elementos en una sola fila o columna (unidimensional).

1. Activar

.contenedor {
  display: flex;
}

2. Ejes

Define si es Fila o Columna.

flex-direction: row | column;

3. Alineación Eje Principal

Usa justify-content.

/* centro, espacio entre, etc. */
justify-content: center | space-between;

4. Alineación Eje Cruzado

Usa align-items.

/* estirar (stretch), centro */
align-items: stretch | center;

5. Comportamiento

/* ¡Estírate! */
flex-grow: 1;

/* Salta de línea si no cabes */
flex-wrap: wrap;

📐 GRID

El Arquitecto: Ideal para estructuras complejas de dos dimensiones (filas + columnas).

1. Activar

.pagina {
  display: grid;
}

2. Definir la Red

Usa px, % o la fracción fr.

/* 3 columnas (la del medio doble) */
grid-template-columns: 1fr 2fr 1fr;

/* Fila superior 100px, resto auto */
grid-template-rows: 100px auto;

3. Mapa Visual

/* Dibuja tu layout con palabras */
grid-template-areas:
  "header header"
  "nav    main"
  "footer footer";

4. Espaciado (Gap)

/* Espacio "mágico" entre celdas */
gap: 20px;

5. Ubicación Mágica

/* Ocupa de columna 1 a la 3 */
grid-column: 1 / 3;

LA REGLA DE ORO DE BIT & VATIO

  • ¿Elementos en línea? ➡️ Flexbox
  • ¿Esqueleto de la web? ➡️ Grid
  • ¿Nivel Pro? ➡️ Flexbox *dentro* de Grid
