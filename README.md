## 🚀 ¿Qué encontrarás en esta guía?

Esta "Hoja de Trucos" (Cheat Sheet) está dividida en las dos metodologías más potentes de CSS moderno:

### 🧩 FLEXBOX (Unidimensional)

Ideal para alinear elementos en una sola fila o columna. El "contorsionista" de CSS.

- **Uso clave:** Barras de navegación, centrado de items, galerías simples.

### 📐 GRID (Bidimensional)

Ideal para estructuras de dos dimensiones (filas y columnas al mismo tiempo). El "arquitecto".

- **Uso clave:** Layouts de páginas completas, dashboards, rejillas complejas.

---

## 💡 La Regla de Oro de Bit & Vatio

> 1. **¿Elementos en línea?** ➡️ Usa **Flexbox**.
> 2. **¿Esqueleto de la web?** ➡️ Usa **Grid**.
> 3. **¿Nivel Pro?** ➡️ Usa **Flexbox** _dentro_ de **Grid**.
